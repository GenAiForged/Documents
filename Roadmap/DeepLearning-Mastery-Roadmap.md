## Core Architecture Progression

### Basic CNN Architectures (2012-2016)
- **AlexNet** (2012) - Breakthrough in ImageNet
- **VGGNet** (2014) - Very deep convolutional networks
- **GoogLeNet/Inception** (2014-2016) - Multi-scale processing
  - Inception v1, v2, v3, v4
  - InceptionResNet - Combining Inception with residual connections
- **ResNet** (2015) - Residual Learning
- **Xception** (2016) - Depthwise separable convolutions
- **MobileNet** (2016) - Efficient networks for mobile
- **DenseNet** (2016) - Dense connections between layers

### Generative Models

**Basic GANs (2014-2016)**
- GAN - Generative Adversarial Networks
- DCGAN - Deep Convolutional GAN
- InfoGAN - Interpretable representations
- Improved Training Techniques

**Advanced GANs (2017-2019)**
- WGAN - Wasserstein GAN
- CycleGAN/DiscoGAN - Unpaired image translation
- Progressive GANs - High-resolution generation
- Style-based Generator Architecture

**Other Generatives**
- Autoencoders & Denoising Autoencoders
- Variational Autoencoders (VAE)
- VQ-VAE - Vector Quantized VAE
- PixelCNN - Autoregressive pixel modeling
- WaveNet - Audio generation

### Advanced Topics

**Normalization Techniques**
- Batch Normalization
- Layer Normalization
- Instance Normalization
- Group Normalization
- Switchable Normalization

**Regularization & Optimization**
- Dropout & Inverted Dropout
- Weight decay
- Learning rate scheduling
- Xavier & Kaiming initialization

**Specialized Learning Paradigms**
- Meta-Learning (MAML, Prototypical Networks)
- Zero-Shot & Few-Shot Learning
- Transfer Learning
- Self-Supervised Learning

**Sequential & Attention Models**
- RNN, LSTM, GRU
- Sequence-to-Sequence
- Attention mechanisms
- Transformer Architecture
- BERT, GPT-2

**Object Detection & Segmentation**
- R-CNN family (Fast R-CNN, Faster R-CNN)
- YOLO - You Only Look Once
- SSD - Single Shot MultiBox Detector
- FCN, U-Net, Mask R-CNN

**Neural Architecture Search (NAS)**
- NAS with RL
- Progressive NAS (PNAS)
- ENAS - Efficient NAS
- DARTS - Differentiable Architecture Search

**Theory & Generalization**
- Loss surfaces and optimization
- Information theory in deep learning
- Lottery Ticket Hypothesis
- Generalization bounds
- Adversarial robustness
